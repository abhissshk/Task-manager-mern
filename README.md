#  Task Manager App (MERN Stack)

A full-stack Task Manager application built using the MERN stack. Users can create, update, delete, and search tasks easily.

---

# Features

- ✅ Add new tasks
- ✏️ Update existing tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as complete/incomplete
- 🔍 Search tasks in real-time
- 🔔 Toast notifications for actions

---

## Tech Stack

### Frontend:
- React.js
- Bootstrap
- React Icons
- React Toastify

### Backend:
- Node.js
- Express.js

### Database:
- MongoDB
